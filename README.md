# Thala 7 Sneaker Store

A simple sneaker store website built using only HTML and CSS.

## Project Files
- `index.html` — contains all the sections of the website (Home, Product, Cart, Checkout)
- `style.css` — contains all the styling like colors, fonts, and layout

## How to Run
Just double-click `index.html` and it will open in your browser. That's it!

## How Navigation Works
All pages are inside one single `index.html` file. Clicking a nav link scrolls you to that section using `id` (like `href="#buy"` scrolls to `id="buy"`).

## Header
The header is fixed for the first 3 pages but it is not fixed for the last checkout-page .
